### 说明
此程序的逻辑来源于微信公众号 - "ABC成本分析"中所分享的文章：
"目标价模拟案例详解" （"产品底价怎么定"系列课程）
这篇文章浅显易懂的讲述了业务场景（卖烧饼）中，如何从
 1. 确立盈利目标
 2. 市场调查（潜在销量，产品类型，市场价格）
 3. 成本解构（固定成本，可变成本）
 4. 确立销量目标
 5. 建立利润表（损益表），现金流量表
 6. 确立并计算项目经济性评估指标：NPV（净现值），IRR（内部收益率）
 7. 分析盈利目标是否达成
 8. 进一步分析项目指标（NPV，IRR）对成本，售价，销量的敏感程度
 9. 根据结果来确定提高盈利的方式：降低成本？提高销量？提高售价？
